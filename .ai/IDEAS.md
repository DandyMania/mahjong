## 土壇場コンボ（ギリギリ大打撃）
**発案**: 2026-06-28  
**概要**: 残りタイムがギリギリ（残り1〜2秒）でコンボ中に正解すると、通常ダメージに加えて特大演出＋ボーナスダメージ

**ユースケース**:
- コンボが続いていて時間がギリギリになってしまった → 焦りながら正解 → 「土壇場逆転！！」的な気持ちよさ
- 残り時間が少ないとリスク感が上がる、それを乗り越えたときの爽快感

**実装案**:
```js
// selectTile の safe ブランチで
const isClutch = _tv <= 2 && G.combo >= 2;
if (isClutch) {
  hpDmg += 1; // ボーナスダメージ
  pts += 150;
  showMoveName('clutch', '土壇場逆転！！');
  // 追加フラッシュ演出
}
```

**CSS**: `.move-clutch` 色は白金〜レインボー系

**懸念点**: comboTimeCarry と相性が良い（ギリギリで正解するとキャリーが少ない→次問題も短い→スパイラル）。緩和策として clutch 成功時にキャリー時間 +3秒？
