# 斬★牌★王 — 仕様書

> 最終更新: 2026-06-28

## ゲームコンセプト

捨て牌ローグライク・バトル。プレイヤーはリーチしたライバルの捨て牌を読んで安全牌を選ぶ。  
正解するとライバルにダメージ、間違えると自分のライフが減る。全ライバルを倒したら完全制覇。

**ターゲット**: 女性ジャンたまプレイヤー、カジュアルに牌の読みを練習したい人

---

## 画面フロー

```
screen-title
  ├─ screen-how (遊び方)
  └─ [スタート] → screen-game
                     ├─ [ライバル撃破] → screen-skill → 次のライバル
                     ├─ [全ライバル撃破] → screen-victory
                     └─ [ライフ0] → screen-gameover → screen-upgrade
                                                          └─ [次のラン/タイトル]
```

## ゲームシステム

### ライバル (RIVALS)

| No | 名前 | HP | 使用問題 ID |
|---|---|---|---|
| 1 | ルーキー あいこ | 4 | 1-5 |
| 2 | ほのか | 5 | 2-7 |
| 3 | 中堅 タケシ | 6 | 5-10 |
| 4 | 強豪 リサ | 7 | 7-13 |
| 5 | 黒龍 ジン | 8 | 9-15 |

問題はシャッフルして使い回す。HP 分だけ正解すればライバル撃破。

### 安全牌判定

| 選択 | 効果 | スコア |
|---|---|---|
| **現物**（捨て牌と同じ牌） | ライバルHP -1、コンボ+1 | 100pt + コンボボーナス |
| **安全牌（非現物）** | ライバルHP -1(or-2)、コンボ+1 | 200pt + コンボボーナス ← **クリティカル** |
| **危険牌** | 自分ライフ-1、コンボ0 | 0pt |
| **タイムアップ** | 自分ライフ-1、コンボ0 | 0pt |

- コンボボーナス = `min(combo, 5) × 20pt`
- クリティカル (非現物安全牌) = ライバルダメージ × rivalDmgMult (デフォルト 1、double_dmg スキルで ×2)
- 現物は必ず安全。非現物安全牌は問題定義に従う

### タイマー

- ベース: 10秒（アップグレード time で +2秒ずつ最大 14秒）
- DEBUG モード: 30秒、EASY モード: 15秒
- **コンボ時間繰り越し** (`carryTime`):
  - クリティカル正解後: `_tv + 4` を次問題の初期時間として使う（最大15秒）
  - 現物正解後: `_tv + 1`
  - ミス/タイムアップ: 繰り越しリセット

### ライフ

- 初期ライフ = `3 + アップグレード hp レベル`（最大 5）
- ライフ 1 でクリティカル演出（赤ビネット + ゾーン背景変更）

## ランダムイベント (`rollEvent`)

各問題開始前に抽選（`G.phase === 'playing'` の時のみ）

| イベント | 確率 | 効果 |
|---|---|---|
| ツモ被害 | `6〜18%`（ライバル進捗に応じて増加） | ライフ-1。鉄壁スキルで防止。幸運お守り Lv2 で無効化 |
| 他家ロン（幸運） | `8%`（幸運お守り Lv1 で ×2） | 次の問題で危険牌 1枚が safe になる (`safeOneNext=true`) |

## EXP・セーブデータ

```js
{ exp: number, upgrades: {[id]: level}, bestScore: number, runs: number }
```

- localStorage キー: `mj-sweeper-v1`
- EXP 獲得 = `max(2, rivalIdx × 3 + floor(score / 50))`
- ゲームオーバー・勝利どちらでも獲得

---

## 永続アップグレード (UPGRADES)

ゲームオーバー後のショップで EXP を消費して購入。ランをまたいで永続。

| ID | 名前 | 最大Lv | コスト | 効果 |
|---|---|---|---|---|
| hp | 体力強化 | 2 | 8, 20 | ライフ +1（最大5まで） |
| time | 時間延長 | 2 | 10, 18 | タイマー +2秒（最大14秒） |
| hint | ヒント全開 | 1 | 15 | ヒントで全安全牌が光る |
| lucky | 幸運お守り | 2 | 12, 25 | Lv1: ラッキーイベント確率×2 / Lv2: ツモ被害無効 |
| combo | コンボキープ | 1 | 20 | ミスでコンボ -1のみ（0にならない） |
| ins | 初回ミス保険 | 1 | 10 | 1ランに1回、最初のミスを無効 |

## ランスキル (RUN_SKILLS)

ライバル撃破後に 3枚から 1枚選択。そのランのみ有効。

| ID | 名前 | 効果 | 実装状態 |
|---|---|---|---|
| iron_wall | 鉄壁 | 次のダメージ（ツモ含む）1回無効 | ✅ |
| time_boost | 速攻 | このライバル戦 タイマー+3秒 | ✅ |
| combo_save | 連撃 | 次のミスでコンボ -1のみ | ✅ |
| safe_one | 幸運の牌 | 次の問題の危険牌1枚が safe になる | ✅ |
| double_dmg | 破壊力 | このライバル戦 与えるダメージ×2 | ✅ |
| score_2x | 大勝負 | 次の正解スコア×2 | ✅ |
| crit_triple | 現物の極意 | クリティカルで与えるダメージ×3 | ✅ |
| heal | 回復 | ライフ+1（即時） | ✅ |
| time_next5 | 余裕 | 次の問題だけタイマー+5秒 | ✅ |
| mouhai | 轟盲牌 | 毎問題、手牌の文字が「？」に。正解で3倍スコア | ✅ |
| reitan | 冷たい打ち手 | ミス/タイムアップでコンボ-1のみ（0にならない） | ✅ |
| shinsoku | 捨て牌三倍速 | 残り5秒以上で正解したらスコア+200 | ✅ |

> **heal** はライフ満タン時は選択肢に出ない。  
> スキル選択画面は 6秒カウントダウン後に先頭（★オススメ表示）を自動選択。

## 問題フォーマット (problems.js)

```js
{
  id: number,              // 1-15
  difficulty: 'beginner' | 'medium' | 'advanced',
  opponentDiscards: ['Nm'], // N=数字, m/p/s/z
  hand: [
    { tile: 'Nm', safe: boolean, reason: string, damage?: number, lucky?: boolean }
  ],
  waits: ['Nm'],
  waitShape: string,       // トースト表示用
  explanation: string
}
```

- `damage`: 省略時 1。2以上でダブルダメージ演出
- `lucky`: `rollEvent` や `safeOneNext` で動的に付与される

## タイル表示 (CSS テキストタイル)

Unicode 絵文字は Windows Chrome で不安定なため CSS 描画を採用。

- 萬子 (m): 赤
- 筒子 (p): 青
- 索子 (s): 緑
- 字牌 (z): 紫/濃紺

**mouhai 発動中**: 手牌が `tile-blind` クラスを持ち「？」表示（紫背景）。捨て牌は通常表示のまま。

## 演出システム

### 技名スラム (`showMoveName`)

| mode | 条件 | 色 |
|---|---|---|
| safe | 通常正解 | 金 |
| critical | 非現物安全牌正解 | 紫 |
| danger | ミス / タイムアップ | 赤 |
| rival | ライバル撃破 / 完全制覇 | 紫 |

### スクリーンフラッシュ

- 正解: 緑フラッシュ
- ミス(1dmg): 赤フラッシュ×2回
- ミス(2dmg): 赤フラッシュ×3回

### コンボ表示

コンボ 2以上で HUD に `🔥 ×N` 表示。テキストバンプアニメーション付き。
