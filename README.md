# 牌★斬★王 (PAI★ZAN★OH)

麻雀マインスイーパー PWA — リーチした対戦相手の捨て牌を読んで安全牌を選ぶ、捨て牌ローグライクバトル。

**プレイ**: https://dandymania.github.io/mahjong/

---

## ゲーム概要

- 相手がリーチ。手牌から **安全牌を 10 秒以内に選ぶ**
- 正解 → 相手に HP ダメージ。失敗 → 自分のライフが減る
- ライバル 5 体を順番に倒したら完全制覇
- ゲームオーバー後に EXP を消費してアップグレード購入 → ループ

**ターゲット**: 女性ジャンたまプレイヤー、カジュアルに牌の読みを練習したい人

---

## 牌読みの基本（ゲーム内教育）

| 理由 | 安全度 |
|---|---|
| **現物**（捨て牌と同じ牌） | 100% 安全 |
| **スジ**（捨て牌 ±3 の牌） | 両面待ち相手なら安全 |
| **字牌**（東南西北白発中） | 比較的安全（確実ではない） |
| **端牌**（1・9 の牌） | 比較的安全（確実ではない） |
| **消去法** | 読みでは判断できない |

正解後のトーストに選んだ牌がなぜ安全だったかが表示される。

## 開発

ビルドステップなし。`index.html` をブラウザで直接開くか静的サーバで配信。

```powershell
python -m http.server 8080
# → http://localhost:8080
```

PWA / Service Worker テストは HTTPS または localhost 必須。

## ファイル構成

```
index.html        — 全画面 (タイトル/遊び方/ゲーム/勝利/ゲームオーバー)
css/style.css     — 全スタイル（モバイルファースト、ダークグリーンテーマ）
js/tiles.js       — TILE_EMOJI / TILE_NAME テーブル
js/problems.js    — PROBLEMS 配列（18問、safeType 付き）
js/game.js        — ゲームロジック + DOM レンダリング
manifest.json     — PWA マニフェスト
sw.js             — Service Worker（キャッシュファースト）
.ai/SPEC.md       — 詳細仕様書
.ai/WISHLIST.md   — 要望・ロードマップ
```
